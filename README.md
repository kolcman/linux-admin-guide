# 🐧 Linux Admin Guide (Go Edition)

Интерактивная TUI-шпаргалка для системных администраторов и DevOps-инженеров. Работает на любом Linux-сервере без установки зависимостей, интерпретаторов или сложных настроек.

### ✨ Почему это удобно?
*   **Портативность:** Один бинарный файл (~4 МБ). Скопировал и запустил.
*   **Работает везде:** Ubuntu, CentOS, Debian, Arch — неважно. Нужен только терминал.
*   **Без GUI:** Идеально для управления серверами через SSH.
*   **Умное копирование:** Быстрое сохранение команд в буфер обмена.
---

### 🚀 Быстрый старт

Скопируй эти команды в терминал своего сервера:

```bash
# 1. Скачиваем последнюю версию
wget https://github.com/kolcman/linux-admin-guide/releases/download/v1.0/linux-guide

# 2. Делаем файл исполняемым
chmod +x linux-guide

# 3. Запускаем гид
./linux-guide
