# 🐧 Linux Admin Guide (Go Edition)

Интерактивная TUI-шпаргалка для системных администраторов и DevOps-инженеров. Работает на любом Linux-сервере без зависимостей и сложной установки.

---

## ✨ Возможности

- 📦 Один бинарный файл (~4–6 MB)
- ⚡ Быстрый запуск через SSH
- 🖥 Работает без GUI
- 🧠 Подсказки по Linux / Docker / Go / Kubernetes
- 📋 Удобная навигация в терминале
- 🔍 Быстрый доступ к командам

---

## 🚀 Быстрый старт

### 1. Скачать последнюю версию
```bash
wget https://github.com/kolcman/linux-admin-guide/releases/latest/download/linux-guide
```
---

### 2. Сделать исполняемым
```bash
chmod +x linux-guide
```
---

### 3. Запустить
```bash
./linux-guide
```
---

## 🧩 Установка в систему (рекомендуется)
```bash
sudo mv linux-guide /usr/local/bin/
linux-guide
```
---

## 🔄 Обновление
```bash
wget -O linux-guide https://github.com/kolcman/linux-admin-guide/releases/latest/download/linux-guide
chmod +x linux-guide
sudo mv linux-guide /usr/local/bin/
```
---

## 📁 Требования

- Linux (x86_64)
- Терминал (SSH или локальный)
- Без дополнительных зависимостей

---

## 🧠 Идея проекта

Быстрый доступ к системным командам и подсказкам прямо из терминала без необходимости искать документацию.
